- [ ] Inventory complete
- [ ] Large files identified

- [ ] Duplicated payloads identified

## Risk

- [ ] Clinical risks identified

- [ ] Build passes

- [ ] Duplication reduced
- [ ] Behavior preserved

## Report

- [ ] Changes documented
- [ ] Risks documented
- [ ] Remaining debt documented
