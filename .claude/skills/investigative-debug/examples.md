# Example

Problem:

Emergency adjustment does not affect calculations.

Wrong Approach:

"Let's add 30%."

Correct Approach:

1. Where is the adjustment selected?
2. Where is it stored?
3. Is it present in state?
4. Is it present in payload?
5. Is it reaching the API?
6. Is it processed by backend logic?
7. Is it returned in the response?
8. Is the UI rendering the updated value?

Only after these questions are answered should a fix be proposed.
